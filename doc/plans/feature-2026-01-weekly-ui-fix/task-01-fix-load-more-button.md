## 実行結果

**ステータス: 完了**

### 実施した変更

**WeeklySummary.razor** の3箇所の「もっと見る」ボタンを修正:

- 変更前: `もっと見る (@(totalCount - articlesCount) 件)` - 全残り件数を表示
- 変更後: `もっと見る（次の @loadCount 件）` - 次に読み込む件数を表示

```razor
@{
    var remaining = totalCount - articlesCount;
    var loadCount = Math.Min(remaining, PageSize);  // PageSize = 10
}
もっと見る（次の @loadCount 件）
```

### 確認結果

- LoadMoreNews/LoadMoreTech/LoadMoreAcademic メソッドは既に PageSize（10件）ずつ追加読み込みするロジックだった
- ボタンラベルのみ修正で対応完了
