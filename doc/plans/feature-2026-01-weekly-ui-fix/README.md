# 実装プラン: 週次おすすめUI修正

## 概要

週次おすすめページの複数のUI問題を修正します：
1. 「もっと見る」ボタンが全件数を表示している問題（10件ずつ表示すべき）
2. ボタンクリック時の例外発生
3. プラスボタンが未認証ユーザーにも表示される問題
4. URLに日本語が入る問題（Slug対応）
